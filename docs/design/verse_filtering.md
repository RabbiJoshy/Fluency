# Per-Artist Verse Filtering

## Decision

Decided against. Not implementing.

## Reasoning

- Goal is understanding full songs — feature verses help learners
- Genius verse labels are unreliable
- Existing filters (English stripping, exclusions, adlib removal) handle real quality problems
