# Alternative Translation Sources for Lyrics

## Decision (April 2025)

Genius + Gemini (~$5/artist) remains best approach for lyric translations.

## What was researched

- **Musixmatch:** Blocked / API access restricted
- **LyricsTranslate:** Too sparse for reggaeton
- **LyricFind:** Enterprise-only, not accessible

## Revisit triggers

- Musixmatch unofficial API stabilizes
- A cheaper/free translation API proves good enough for lyrics (see `docs/design/translation_services.md` for active research on this)
